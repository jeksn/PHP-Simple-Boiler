# PHP Simple Boiler

Start dev server
```
php -S localhost:8080
```

Run Tailwind watch on files
```
./tailwindcss -i ./dist/input.css -o ./dist/output.css --watch
```

Minify for production
```
./tailwindcss -i ./dist/input.css -o ./dist/output.css --minify
```
